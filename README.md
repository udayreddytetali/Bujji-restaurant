# Bujji Restaurant Website with Billing Calculator

A clean, responsive restaurant landing page built with HTML, CSS, and JavaScript featuring an interactive billing calculator.

## Features

- **Hero Section:** Large welcoming banner with background image and call-to-action.
- **Menu Section:** Displays popular dishes with images, names, and prices.
- **Bill Calculator:**  
  - Add, remove, and adjust menu items dynamically  
  - Instant calculation of subtotal, tax, tip, discount, and total  
  - User-friendly inputs for price, quantity, and tax percentage
- **Responsive Design:** Looks great on desktop, tablet, and mobile devices.
- **Pure Vanilla JS and CSS:** No frameworks for easy customization.

## Getting Started

Simply clone or download the repository, open the `index.html` file in a modern browser, and start exploring the menu and billing features.

## Technologies Used

- HTML5
- CSS3 (with CSS variables for theme colors)
- JavaScript (ES6), DOM manipulation for interactive bill calculator

## How to Use

- Browse the menu section to see dishes offered.
- Use the billing calculator at the bottom to simulate ordering by adjusting item quantities, prices, and taxes.
- Add new items or remove existing ones to customize your bill.
